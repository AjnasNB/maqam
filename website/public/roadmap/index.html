Release lines

Maqam 0.3.1 + ProductLoop OS 0.2.1

Maqam's 0.3 release line expands governed research and structural browser actions without claiming that the registry itself is a browser engine, crawler fleet, authentication system, or hosted control plane.

  • Governed source contract

    Static source adapters, required ToolCaller routing, explicit-unavailability fallback, fail-closed unknown errors, normalized records, and bounded host-defined checks are Maqam 0.3 APIs.

  • Feed-aware crawler

    RSS and Atom parsing and discovery operate inside the crawler's explicit origin, request, depth, byte, redirect, robots, and duration controls.

  • ProductLoop package family

    ProductLoop composes versioned umbrella, Maqam boundary, runtime, policy, approvals, provenance, skills, connectors, evaluations, and bounded browser-research packages; verify each named version on npm before installation.

Next, no date promised

Reproduction and integration hardening

  • Independent MGES reruns

    Invite Linux and additional host reproductions while keeping environment-specific results separate.

  • Offline provider fixtures

    Test explicit Google ADK and Microsoft Agent 365 seams without credentials, model calls, or live effects.

  • Reviewed source adapters

    Add provider or capability-specific fixtures only after their authentication, failure, data, and bypass boundaries are documented and tested.

  • Durable state contract

    Specify integrity, transactions, concurrency, migration, retention, and reviewer authentication before claiming durable governance.
