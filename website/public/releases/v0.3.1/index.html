Governed browser boundary

Structural operations, exact authority.

Maqam registers four bounded operations around a host-owned driver. It does not bundle a browser engine, account session, credential store, or arbitrary-script executor.

  • Observe

    Return a bounded structural page record from one explicitly allowed HTTP or HTTPS origin.

  • Preview

    Create an opaque same-run plan without accepting raw values, selectors, script, or hidden navigation authority.

  • Apply

    Require the exact approval action, exact plan, active dispatch guard, and re-observation before mutation.

  • Submit

    Bind expected origin and new-page behavior, then require fresh final evidence from the resulting page.

Governed public research

Useful reach without pretending the internet has no gates.

  • Public YouTube metadata

    An explicitly configured external yt-dlp executable can retrieve bounded public metadata and available captions without media downloads, cookies, or login bypass.

  • RSS and Atom

    Offline parsing, safe link resolution, bounded text, feed discovery, and evidence-linked normalized records.

  • Bounded crawling

    Origin, DNS, redirect, robots, request, byte, depth, duration, feed, and callback limits remain explicit.

  • Optional hosted search

    An Exa MCP adapter can be enabled separately. Authentication and provider availability remain explicit instead of silently falling through.
