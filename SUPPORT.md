# Maqam support

Use the right channel so questions remain searchable and security reports remain private.

- Usage, architecture, and integration questions: GitHub Discussions.
- Reproducible defects: GitHub Issues using the bug form.
- Feature and adapter proposals: GitHub Discussions first, then a feature issue when the scope is concrete.
- Security vulnerabilities: follow [SECURITY.md](SECURITY.md).
- npm release or provenance problems: open a bug report and include the package version, Node.js version, operating system, install command, and non-sensitive registry output.

Maqam is community-supported. No response-time or resolution SLA is provided.
