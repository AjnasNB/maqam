# License Audit

Maqam is an original Ajnas implementation released under MIT.

## Direct Runtime Dependencies

- `cheerio`: MIT-compatible HTML parsing dependency.
- `robots-parser`: MIT robots.txt parsing dependency.
- `turndown`: MIT HTML-to-Markdown conversion dependency.

Resolved transitive dependency licenses are recorded in `package-lock.json`.

## Inspected Inspiration

- Qwen-Agent: https://github.com/QwenLM/Qwen-Agent, Apache-2.0. Reviewed for public agent/tool/model separation patterns. No source code, docs, examples, or branding were copied.
- PageAgent: https://github.com/alibaba/page-agent, Apache-2.0. Reviewed for public page-agent boundary patterns. No source code, docs, examples, or branding were copied.

## Boundary

Public permissive projects may be cloned or installed only for inspection, benchmarking, and learning. Maqam source, docs, examples, API names, release gate behavior, approval queue behavior, and package identity are original Ajnas work.

No third-party source code, private content, gated content, proprietary identity, logos, or project branding is included in this release candidate.
